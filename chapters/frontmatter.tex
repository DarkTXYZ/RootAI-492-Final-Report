\begin{abstract}
The board game called ``Root'' is a partially observable asymmetric game where each player has different game mechanics and goals. There are many factions within Root, each having its own way to achieve the victory condition of accumulating 30 ``victory points (VP)''. This project focuses on 2 base factions: ``Marquise'' de Cat and the ``Eyrie'' Dynasties.

We developed an artificial intelligence (AI) system capable of playing Root. The AI agent can take game state as input and make intelligent decisions in each phase.

The main algorithm for implementing the AI is Monte Carlo Tree Search (MCTS). MCTS algorithm maximizes the probability of winning by simulating many rollouts and choosing the best decision.

There are 6 parameters for the our MCTS algorithm: \texttt{reward-function}, \texttt{expand-count}, \texttt{rollout-no}, \texttt{time-limit}, \texttt{action-count-limit}, and \texttt{best-action-policy}. By altering these parameters, we created 108 MCTS agent variants for each faction.

The 108 agents are pitched against the other faction with the ``base'' variant. The top 5 variants with the highest win rate for each faction are selected. The 5 variants from each faction all play against each of the other faction's. The best variant for each faction is then selected.

The best MCTS variant for \Marquise{} have the following parameters respective to the above parameter list: \texttt{vp-difference-relu}, \texttt{200}, \texttt{1}, \texttt{-1 (no limit)}, \texttt{-1 (no limit)}, and \texttt{secure}. With average win rate against 5 \Eyrie{} agents of 60\%.

The best MCTS variant for \Eyrie{} have the following parameters respective to the above parameter list: \texttt{vp-difference-relu}, \texttt{200}, \texttt{1}, \texttt{-1 (no limit)}, \texttt{-1 (no limit)}, and \texttt{secure}. With average win rate against 5 \Marquise{} agents of 60\%.
